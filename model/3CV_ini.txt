;############# this TestCircuit is written to support both, LIST/LIN and LSYNC sweeps of vd, vg ##########
; below the special solution if non-rectangular measurement stimuli had been applied (LSYNC sweeps of vd and vg)
; => add a series voltage to each of the Input voltages vd_aux and vg_aux, controlled by the LSYNC Inputs vd and vg, and set these two Inputs set to Mode CON and 0V
; Note: if standard Inputs for vd and vg are used (LIN, LIST Mode), the 2 add'l voltage sources below don't matter.

; Since the losses affect only the DC path, they are shorted

R:Rbias2 P22 P222 R=0.4
C:CACshort2 P22 P222 C=(.1)
